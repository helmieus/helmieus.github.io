### Bitcoin wallet ₿

Lastly, we cat the `ine.conf` file and got something interesting.

![obfuscated string](/assets/img/posts/2026-01-26-TryHack3M/miner.png)

We got an obfuscated string... So we go to CyberChef to decode it.

![cyberchef](/assets/img/posts/2026-01-26-TryHack3M/cyberchef.png)

I used 'magic' function to decode it automatically. In the output field, there are two *bitcoin wallet addresses*.

```
bc1qyk79fcp9hd5kreprce89tkh4wrtl8avt4l67qa
bc1qyk79fcp9had5kreprce89tkh4wrtl8avt4l67qa
```

So we used [blockchain.com](https://www.blockchain.com/explorer/) to check if the addresses are valid.

![check wallet](/assets/img/posts/2026-01-26-TryHack3M/check_wallet.png)

We found the first address is valid.
